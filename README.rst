Install
------------------------------------------------------------------------------

``aws_lambda_event`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install aws_lambda_event

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade aws_lambda_event
